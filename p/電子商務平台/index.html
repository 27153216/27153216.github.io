ASP.NET SQL Server

電子商務平台

課堂作品/ 購物網站

Jun 02, 2019

https://github.com/27153216/EC-Project

作品介紹

這是我在電子商務的課程中做的課堂作品，由 ASP.NET 結合 SQL Server 開發的電子商務平台。
本專題為小組開發，網頁與程式部分全部由我開發，小組成員負責商品構想、行銷計畫、價格制定等等其他部分。
此購物網站原先是連結到學校的資料庫，但由於課程結束，資料庫已經關閉，因此這個版本改成本地資料庫。

功能介紹

首頁由五張輪播圖構成。

品牌故事介紹。

購買商品頁面。

（商品的資料為方便演示，並非當時的真實資料） 購買按鈕已經沒有功能，因為學校伺服器已經關閉。

帳號註冊、登入。

登入後可以在右上方看見使用者名稱。
此時由於我登入的是我的管理帳號，所以可以看到管理的按鈕。

因為在不斷修改資料的過程中，每次修改都要連入學校的資料庫，打一長串資料庫指令，非常繁瑣。
因此我自己建立了一個管理頁面，並且要登入指定的帳戶才能進入，方便我自己修改及新增資料。
