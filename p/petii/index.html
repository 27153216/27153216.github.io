作品介紹

這是我在資料庫應用程式設計的課程中做的課堂作品，一款用 Windows Form 開發的寵物小遊戲。
語言使用 C# ，結合 SQL Server 儲存玩家資料。
這個作品為 3 人小組開發，我的負責部分為程式撰寫、優化，後半段寵物互動環節的所有內容。

功能介紹

開啟程式後，可以在畫面中開啟新遊戲或讀取舊有紀錄，旁邊可以調整音量。

最多可以有三個存檔紀錄。

開始新遊戲後，會進入一段前置劇情，並幫寵物取名。

前置劇情結束後，進入寵物互動環節。（由此開始是我獨立開發）

畫面中會有一隻貓咪四處走動，玩家可以點擊右方按鈕與她進行各種互動。
畫面上方可以看到操作的紀錄和親密度。

此處我們點擊右上方的飼料，會看見飼料隨著滑鼠移動。
放置於地板後，貓咪會衝向飼料，並增加親密度。
若放置於無法取得的地方，貓咪會提醒玩家。

玩家也可以碰觸貓咪，增加親密度，左上方可以看見玩家操作紀錄。

玩家若離開程式，下次開啟時也可以讀取上次遊玩的進度。
